задача: создать телеграмм бот с базой данных  PostgreSQL. Бот выводит пользователям набор данных из базы данных и позволяет производить поиск. База данных наполняется импортом из excel файла. Все шаги делаем поэтапно. Работаю на windows  в VC Code. Начинаем с анализа структуры excel файлов.

В таблице data есть колонка "код магазина" к каждому коду магазина присвоены значения в таблице адрес по коду. Таблица "город_область_округ" нужна для фильтрации по регионам.
